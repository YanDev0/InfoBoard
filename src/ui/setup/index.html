InfoBoard adalah sebuah aplikasi gratis dan sumber terbuka untuk kebutuhan papan jadwal digital di instansi pemerintahan atau instansi lainnya. Perlu diingat bahwa aplikasi ini dibuat TANPA GARANSI apapun. Seluruh kesalahan yang terjadi merupakan tanggung jawab pengguna. Masalah aplikasi dapat dilaporkan di halaman issue Github.
